Class: NamedEntityRecognitionNode

NamedEntityRecognitionNode(name, text, labels)

A service configuration for NamedEntityRecognitionService for Pipeline use.

Constructor

new NamedEntityRecognitionNode(name, text, labels)

Parameters:
Name Type Description
name string The name of this Node. It will be used by the Pipeline to reference this Node.
text string Input text to be analyzed for named entities.
labels Object.<string, string> When providing labels, the module will extract entities that match your labels and descriptions. This gives enough flexibility to deal with any use-case.
