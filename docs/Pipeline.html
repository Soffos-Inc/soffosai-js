Class: Pipeline

Pipeline(nodes, use_defaultsopt, nameopt, kwargsopt)

A controller for consuming multiple Services called Nodes. It validates all inputs of all Nodes before sending the first Soffos API request to ensure that the Pipeline will not waste credits. ** use_defaults=true means that Nodes will take input from the previous Nodes' output of the same field name prioritizing the latest Node's output. If the previous Nodes does not have it, it will take from the pipeline's user_input. Also, the Nodes will only be supplied with the required fields + default of the require_one_of_choices fields.
